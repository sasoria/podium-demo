# Podium-demo

A simple demo of a micro-frontend architecture using [Podium](https://podium-lib.io/).

## Features

- Server-side composition of microfrontends
- Server-side routing that can consist of one or more podlets

## Available Scripts

### Layout

```
$ npm run build
$ npm run layout
```

### Podlets

```
$ npm run build
$ npm run podlet
```
